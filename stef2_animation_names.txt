VERSION: 1.0, released 2007.31.12[Germany] by Chrissstrahl

This list contains a selection of animation names for several NPC's as well as for the Players in Multiplayer of Star Trek: Elite Force II.

More animation names can be found in /base/pak20.pk3/models/sharedanim/.
This list contains the most basic animation names.

By dissabling the ai of a npc (script $npcTargetName ai_off) you can make the
npc playing animations. You can use the following commands to animate a npc in
the game console:
script $npcTargetName anim <animationName>
script $npcTargetName animateonce <animationName>

This will stop all animations:
script $npcTargetName stopanimating
